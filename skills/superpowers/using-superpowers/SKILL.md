# Using Superpowers

Skills are operational requirements, not optional reading.

1. Search the normal Sidekick Skills list for a matching workflow before acting.
2. Load the relevant skill and follow its gates before tools or code.
3. Common sequence: `brainstorming` → `writing-plans` → `using-git-worktrees` → `test-driven-development` → `verification-before-completion`.
4. Use Plan Mode for design and inspection; use Execute only after the current plan version is explicitly approved.

When more than one skill applies, use the smallest set that covers the task and state the order.
