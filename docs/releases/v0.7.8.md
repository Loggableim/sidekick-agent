# Sidekick v0.7.8 — Node.js Auto-Install + Corrupt Git Config Fix

**Tag:** `v0.7.8`
**Datum:** 2026-06-08

---

## Ziel des Releases

Zwei Windows Clean-Install-Blocker behoben: Node.js wird jetzt automatisch
installiert (winget → binary download) statt nur zu warnen, und eine
korrupte `.gitconfig` blockiert den Installer nicht mehr.

## Änderungen

### Node.js: Info → Auto-Install

`Test-Node` wurde von einer rein informativen Warnung zu einer Download-
Funktion aufgewertet:

| Stufe | Pfad | Beschreibung |
|-------|------|-------------|
| 1 | `Get-Command node` | Bereits installiert → OK |
| 2 | `$SidekickHome\node\node.exe` | Sidekick-managed → OK |
| 3 | `winget install OpenJS.NodeJS.LTS` | Automatisch (silent) |
| 4 | Binary-Download | Node.js 22 LTS als ZIP → `%LOCALAPPDATA%\sidekick\node\` |
| 5 | Warnung | Beide Pfade fehlgeschlagen → Hinweis |

### Git: Robuste Version-Prüfung

`Install-Git` prüft jetzt `$LASTEXITCODE` und `$version -match "git version"`
bevor ein Git als funktionsfähig akzeptiert wird. Eine korrupte `.gitconfig`
löst keinen Crash mehr aus — der Installer fällt auf PortableGit zurück.
